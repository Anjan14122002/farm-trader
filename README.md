
## ⚙️ Setup Instructions

1. Clone the repo:
   ```bash
   git clone https://github.com/your-username/agrotrader.git
   cd agrotrader
Install backend dependencies:
npm install
Create a .env file:
MONGO_URL=your_mongodb_connection_string
JWT_SECRET_KEY=your_secret
JWT_EXPIRE=5d
Start the server:
npm start
